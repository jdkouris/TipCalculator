# TipCalculator

Source code for the `Build a Tip Calculator` [video](https://youtu.be/fnHDbTDPu3s) 
on my Youtube [channel](https://www.youtube.com/c/JohnKourisDev)

## Things You'll Learn
- Creating a custom counter view that can be ported to other projects
- Working with @Binding, @State, and @FocusState
- Formatting numbers with specifiers
- Styling views based on device size using GeometryReader
- Implementing pickers and how to customize their appearance
